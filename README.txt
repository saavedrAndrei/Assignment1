Dataset Description:
The "Mobile apps to fight the COVID-19 crisis" introduces the different apps published around the world to fight the impact of COVID-19 crisis. The dataset is structured in different features about their status, last update, geographic coverage and many more. Also, a file with all the documentation of every variable or feature is included.
